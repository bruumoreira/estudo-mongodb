# Estudo com MongoDB

## Nome
Estudo com MongoDB

## Descrição
Projeto realizado com foco de estudo no MongoDB com Spring, <u>**não levando em consideração o tratamento de excessão e testes unitários.**</u>

## Funcionalidades
- Desenvolver um sistema de controle de filmes, onde seja possível criar, alterar, buscar e excluir.

## Tecnologias
- Docker
- Java
    - SpringBoot
- MongoDB
- Swagger

## Inicialização
    1 - É necessário startar o banco. Utilizando o docker-compose do projeto.
    2 - Crie um usuário no banco com login admin_movie e senha 1234 ou altere no arquivo application.properties com os dados de sua preferência.
    3 - Baixe o repositório e abra o arquivo MovieApplication
